20 exercices pour apprendre et se familiariser avec les fonctions en JavaScript ! Fonctions en JavaScript 1. *Fonction simple* : Écris une fonction `saluer(nom)` qui affiche "Bonjour, nom !". 2. *Fonction avec return* : Écris `somme(a, b)` qui retourne `a + b`. 3. *Fonction anonyme* : Utilise une fonction anonyme pour doubler un nombre. 4. *Fonction fléchée* : Réécris `function double(x) { return x * 2; }` en arrow function. 5. *Paramètres par défaut* : Écris `function greet(nom = 'invité') { ... }`. 6. *Fonction callback* : Écris `executer(fn)` qui appelle `fn()`. 7. *Fonction d'ordre supérieur* : Écris `appliquer(fn, val)` qui retourne `fn(val)`. 8. *Fonction récursive* : Écris `factoriel(n)` qui calcule `n!`. 9. *Fonction avec scope* : Explique la portée d'une variable dans une fonction. 10. *Fonction pure* : Écris une fonction pure `somme(a, b)`. 11. *Fonction impure* : Écris une fonction impure `ajouterAuCompteur()`. 12. *Utiliser .bind()* : Lie une fonction à un objet avec `bind`. 13. *Fonction avec paramètres rest* : `function somme(...nombres) { ... }` 14. *Fonction qui modifie un objet* : `function ajouterProp(obj, key, val) { obj[key] = val; }` 15. *Fonction qui clone un objet* : `function clone(obj) { return { ...obj }; }` 16. *Fonction avec promesse* : `function attendre(ms) { return new Promise(...); }` 17. *Fonction async/await* : Écris `async function fetchData() { ... }` 18. *Fonction curryifiée* : Transforme `somme(a, b)` en `somme(a)(b)`. 19. *Fonction avec gestion d'erreurs* : `function diviser(a, b) { if (b === 0) throw Error('...'); }` 20. *Fonction IIFE* : `(function() { console.log('ok'); })()`
